- Use `xmetric.run` when the user references the CSV metric table, asks for scaling, groupby,
  or time-series visualization.
- Use `ymetric.run` when the user mentions scatter plots, correlations, factor pairs, or
  comparisons between two numeric fields.
- Use `portfolio.normalize` when the user uploads allocations, mentions weights summing to
  more/less than 100%, or requests validation of tickers.
- File Search should precede web search unless the query is obviously out-of-corpus or needs
  breaking news. Cite corpus filenames inline.
